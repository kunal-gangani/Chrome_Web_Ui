# Google-like Search Engine

## Description

This project is a simple implementation of a Google-like search engine that allows users to set Username and password. The goal is to provide a basic understanding of how search engines work and to demonstrate web development skills thorugh flutter development.


## Technologies Used

- Flutter
- Dart
